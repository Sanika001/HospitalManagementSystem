# Hospital Management System
A Django-based Hospital Management System that helps manage doctors, patients, and appointments efficiently.
## Features
- Admin Login
- Doctor Management
- Patient Management
- Appointment Booking
- Authentication System
- Bootstrap Frontend UI
## Technologies Used
- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap
## How to Run Project
1. Create Virtual Environment
python -m venv venv
2. Activate Virtual Environment
venv\Scripts\activate
3. Install Django
pip install django
4. Run Server
python manage.py runserver
